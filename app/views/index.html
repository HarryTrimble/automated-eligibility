Automated eligibility

Two teams on GDS' Government Data Exchange (GDX) programme spent several weeks exploring different ways automated eligibility checking between public services could work.

The aims of the prototypes were:

  • Help ask better questions about the value propositon of a GDX platform
  • Focus interviews with public sector organisatiuons about eligibility on opportunities and details

The prototypes are of two fictional software platforms called GOV.UK Proof and Life Events. Both prototypes uses scenarios where data is shared between different public sector organisations reduce the burdens and costs of eligibility checking.

View prototype code on GitHub
